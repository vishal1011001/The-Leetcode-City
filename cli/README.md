# leetcode-city

> One-command setup for [The Leetcode City](https://theleetcodecity.tech) — your LeetCode profile as a 3D pixel art building.

## Quick Start

```bash
npx leetcode-city init
cd The-Leetcode-City
npm run dev
```

That's it. **No API keys needed** for frontend development.

## Commands

| Command | Description |
|---------|-------------|
| `npx leetcode-city init` | Clone the repo, install deps, configure env |
| `npx leetcode-city init my-folder` | Clone into a custom folder name |
| `npx leetcode-city doctor` | Check if your setup is healthy |

## What Works Without Keys

| ✅ Works | ⚠️ Needs service key |
|----------|---------------------|
| View the 3D city | Sign in / auth |
| Browse developer profiles | Claiming buildings |
| UI/CSS/component changes | Shop purchases |
| 3D rendering & animations | Raids & interactions |
| Leaderboard & search | API route writes |

- Node.js 24+
