# Changelog

## 0.2.3

- **Auto-Updater:** Extension now checks GitHub for newer versions on startup (debounced to once every 6 hours) and prompts with a download link
- **Icon Fix:** Activity Bar icon now uses `currentColor` instead of hardcoded colors, fixing invisible icons on certain VS Code themes

## 0.2.0

- Rebranded to **Leetcode City: Pulse**
- New status bar states: Transmitting, Standby, Off, Disconnected
- Updated commands: Pulse: Connect, Disconnect, Toggle, Open City
- Fixed API URL to use www subdomain

## 0.1.0

- Initial release
- Live presence: your building glows when you code
- Privacy controls: choose what data to share
- Status bar integration
- Idle detection with configurable timeout
